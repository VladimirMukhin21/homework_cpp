## Пара 2: Наследование и полиморфизм

### Теоретический материал (кратко)

#### Наследование

```cpp
// Базовый класс
class Shape {
protected:
    int x, y;
public:
    Shape(int x, int y) : x(x), y(y) {}
    virtual void draw() {
        std::cout << "Drawing shape" << std::endl;
    }
    virtual ~Shape() {}
};

// Производный класс
class Circle : public Shape {
    int radius;
public:
    Circle(int x, int y, int r) : Shape(x, y), radius(r) {}

    void draw() override {
        std::cout << "Drawing circle at (" << x << ", " << y
                  << ") with radius " << radius << std::endl;
    }
};
```

#### Виртуальные функции

```cpp
Shape* shape = new Circle(10, 20, 5);
shape->draw();  // Вызовется Circle::draw() благодаря virtual
delete shape;   // Вызовется правильный деструктор
```
